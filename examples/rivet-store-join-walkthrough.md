# Rivet Store Join Line Walkthrough

This note is the quickest way to read the extra review model in `rivet-store-join-line`.

| Case | Focus | Score | Lane |
| --- | --- | ---: | --- |
| baseline | index fit | 188 | ship |

| edge | constraint risk | 195 | ship |

Start with `stale` and `recovery`. They create the widest contrast in this repository's fixture set, which makes them better review anchors than the middle cases.

The useful comparison is `index fit` against `plan drift`, not the raw score alone.
